# 安装环境

​ nodeJS、npm（node 里面有包含）、[git]https://git-scm.com/)

​ 安装环境最好有教程跟着走，需要配置环境，以及一些国内下载速度过慢的问题。

# 安装 Hexo 和主题

​ hexo 官网文档有详细解析

​ 安装： npm install -g hexo-cli

​ hexo 的基本使用，
​ hexo init "xxx" /创建hexo的文件夹

​ cd “xxx” /进入创建文件夹
​ npm install /安装hexo的环境依赖

​ 创建过后，生成的 themes 文件夹就是主题，选用的主题可以在官网挑选。这里的主题选用的是 Shoka。

​ cd “xxx”

​ git clone https://github.com/amehime/hexo-theme-shoka.git ./themes/shoka /从GitHub拉取主题配置

​ 应用主题之前，必须至少安装 hexo-renderer-multi-markdown-it 渲染插件 和 hexo-autoprefixer，详细可以在作者地址中获取安装细节。
